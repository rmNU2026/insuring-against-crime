# Insuring Against Crime: The Long Arm of Childhood Medicaid Eligibility

Replication code for my senior thesis at Northwestern University (Weinberg College of Arts & Sciences, 2026), advised by Professor Frank Limbrock.

## Overview

This paper estimates the causal effect of childhood Medicaid eligibility on adult criminal behavior and tests three mechanism channels — human capital, financial stability, and behavioral health — across five national datasets.

## Data Sources

Raw data must be downloaded separately:

- **FBI Uniform Crime Reports (UCR):** [ICPSR](https://www.icpsr.umich.edu/)
- **IPUMS ACS:** [IPUMS USA](https://usa.ipums.org/)
- **IPUMS CPS ASEC:** [IPUMS CPS](https://cps.ipums.org/)
- **BRFSS:** [CDC](https://www.cdc.gov/brfss/)
- **SAMHSA TEDS-A:** [SAMHSA](https://www.samhsa.gov/data/data-we-collect/teds/datafiles)
- **Simulated eligibility:** Constructed using the BKL calculator applied to March 1996 CPS
